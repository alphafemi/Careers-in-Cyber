# Careers-in-Cyber

Learn about the different careers in cyber security.

Want a career in cyber security? Explore what that could look like for you. Cybersecurity has many career paths, from breaking into systems (offensive security) to defending organisations from attacks (defensive security).

If you enjoy solving problems, being curious, or understanding how things work, you'll likely find a role that fits you.

## Career Paths Overview

### Offensive Security
Breaking into systems to find vulnerabilities before the bad guys do.

- **Penetration Tester** - Simulates attacks to find security weaknesses
- **Red Team Operator** - Conducts comprehensive adversarial simulations
- **Security Researcher** - Discovers and analyzes new vulnerabilities and exploits

### Defensive Security
Protecting organizations from attacks and incidents.

- **Security Analyst** - Monitors systems and responds to security alerts
- **SOC (Security Operations Center) Analyst** - First line of defense, monitors networks 24/7
- **Security Engineer** - Designs and implements security solutions
- **Incident Response Specialist** - Responds to and investigates security breaches
- **Security Architect** - Designs comprehensive security strategies

### Specialized Roles

- **Forensic Analyst** - Investigates cybercrimes and recovers digital evidence
- **Malware Analyst** - Analyzes malicious code to understand threats
- **Cloud Security Engineer** - Secures cloud infrastructure and applications
- **Identity & Access Management (IAM) Specialist** - Manages user access and authentication
- **Compliance Analyst** - Ensures organizations meet security regulations

## Required Skills & Certifications

### Core Technical Skills

- **Networking** - Understanding TCP/IP, DNS, firewalls, and network protocols
- **Operating Systems** - Linux, Windows, and macOS administration
- **Programming/Scripting** - Python, Bash, PowerShell, JavaScript
- **Databases** - SQL and database security concepts
- **Cryptography** - Understanding encryption and cryptographic concepts
- **Web Technologies** - HTML, HTTP, APIs, and web vulnerabilities
- **Cloud Platforms** - AWS, Azure, or Google Cloud security

### Popular Certifications

| Certification | Level | Provider | Best For |
|---|---|---|---|
| CompTIA Security+ | Entry/Mid | CompTIA | Entry-level, foundational knowledge |
| Certified Ethical Hacker (CEH) | Mid | EC-Council | Penetration testing, offensive security |
| CISSP | Advanced | (ISC)¬≤ | Senior roles, management |

| Offensive Security Certified Professional (OSCP) | Mid/Advanced | Offensive Security | Hands-on penetration testing |

| AWS Certified Security - Specialty | Mid | Amazon | Cloud security |

## Getting Started / Learning Path

### Phase 1: Foundation (0-3 months)
1. Learn basic networking and operating systems
2. Understand cybersecurity fundamentals
3. Start with CompTIA A+, Network+, or Security+ studies
4. Set up a home lab for hands-on practice

### Phase 2: Core Knowledge (3-6 months)
1. Deep dive into your area of interest (offensive or defensive)
2. Learn a programming language (Python recommended)
3. Complete beginner CTF (Capture The Flag) challenges
4. Complete your first certification

### Phase 3: Practical Experience (6-12 months)
1. Build projects and contribute to open-source security projects
2. Participate in bug bounty programs
3. Complete internships or entry-level positions
4. Continue certifications based on career path

### Recommended Learning Platforms
- **TryHackMe** - Interactive cybersecurity training
- **HackTheBox** - Hands-on hacking practice
- **Coursera** - University-level courses and certifications
- **edX** - Open-source cybersecurity courses
- **SecurityTube** - Video-based security training
- **PentesterLab** - Practical penetration testing training

## Educational Requirements

### University Degree Path
- **Duration**: 3-4 years
- **Programs**: Cybersecurity, Computer Science, Information Security
- **Pros**: Comprehensive education, networking opportunities, internship programs
- **Cons**: Time and cost-intensive

### Bootcamp Path
- **Duration**: 8-16 weeks
- **Programs**: Cybersecurity bootcamps, coding bootcamps with security focus
- **Pros**: Quick entry, job-focused curriculum, career support
- **Cons**: Intensive, less comprehensive than degrees

### Self-Taught Path
- **Duration**: 6 months to 2+ years (varies)
- **Resources**: Online courses, certifications, books, hands-on labs
- **Pros**: Flexible, cost-effective, self-paced
- **Cons**: Requires discipline and motivation, no formal credential

### Hybrid Approach (Recommended)
- Start with foundational online courses
- Earn entry-level certifications (Security+, CEH)
- Gain practical experience through internships or junior roles
- Pursue advanced certifications or degree as needed

## Salary & Job Market Information

### Average Salaries by Role (US Data, 2024-2025)

| Role | Entry Level | Mid Level | Senior Level |
|---|---|---|---|
| Security Analyst | $55,000 - $70,000 | $75,000 - $95,000 | $100,000+ |
| SOC Analyst | $50,000 - $65,000 | $70,000 - $90,000 | $95,000+ |
| Penetration Tester | $65,000 - $85,000 | $90,000 - $130,000 | $140,000+ |
| Security Engineer | $80,000 - $100,000 | $110,000 - $150,000 | $160,000+ |
| Security Architect | $100,000 - $130,000 | $140,000 - $180,000 | $200,000+ |
| CISO | $150,000 - $200,000 | $200,000 - $300,000+ | $300,000+ |

### Job Market Outlook
- **Growth Rate**: 13% (faster than average)
- **Demand**: High and increasing
- **Geographic Hot Spots**: Silicon Valley, New York, Washington DC, Austin, Seattle
- **Remote Opportunities**: Abundant, especially post-2020

### Factors Affecting Salary
- Certifications and credentials
- Years of experience
- Technical skills and specialization
- Geographic location
- Company size and industry
- Security clearances (government roles)

## Real-World Profiles

### Profile 1: Alex - SOC Analyst to Security Manager

**Background**: Started with CompTIA Security+ certification, no prior IT experience

**Journey**:
- Year 1: SOC Analyst at startup - $55,000
- Year 3: Senior Analyst with CISSP - $85,000
- Year 5: Security Manager - $120,000

**Key Takeaways**: "Consistency and continuous learning were crucial. I spent time every week learning new tools and improving my skills."

### Profile 2: Jordan - Self-Taught Penetration Tester

**Background**: Computer science degree, self-taught security skills

**Journey**:
- Year 0: Self-study, HackTheBox, bug bounties
- Year 1: Junior Penetration Tester - $70,000
- Year 3: Senior Penetration Tester with OSCP - $110,000

**Key Takeaways**: "Hands-on experience through CTFs and bug bounties was more valuable than any course. Build a portfolio!"

### Profile 3: Casey - Cloud Security Specialist

**Background**: Systems Administrator who transitioned to security

**Journey**:
- Year 1: Systems Admin with AWS basics
- Year 2: Cloud Security Engineer with AWS Security cert - $95,000
- Year 4: Senior Cloud Security Architect - $150,000

**Key Takeaways**: "Understanding infrastructure deeply before specializing in security made the transition smooth."
